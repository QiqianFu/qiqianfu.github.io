● 好的，这道题要求找到数组中第k大的元素。有几种经典解法：
方法一：排序（最简单直接）
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        nums.sort()
        return nums[-k]
时间复杂度：O(n log n)，空间复杂度：O(1)
方法二：使用堆（优先队列）
import heapq

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        return heapq.nlargest(k, nums)[-1]
时间复杂度：O(n log k)，推荐面试使用 ✨
